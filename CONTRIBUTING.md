# Contributing to Chess

Thanks for your interest in contributing! Contributions are welcome via issues and pull requests.

How to contribute
- Fork the repository and create a feature branch.
- Make changes on your branch and add clear commit messages.
- Run the project locally and ensure changes compile.
- Open a pull request describing the change and rationale.

Coding style
- Follow existing project style (basic C++ conventions, 4-space indentation).
- Keep changes focused and small; document larger design decisions in the PR.

Testing
- Build the solution locally and verify the application runs.

Contact
- Open an issue for questions or design discussions.
